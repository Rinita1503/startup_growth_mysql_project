# startup_growth_mysql_project
This project analyzes startup growth and investment trends using MySQL. The dataset contains 5,000 startup records across multiple industries and countries, including investment amounts, valuations, funding rounds, investor counts, growth rates, and founding years.

The main objective of this project is to analyze startup funding trends, investment performance, valuation growth, and investor behavior using SQL.
This project is designed as a portfolio project for aspiring Data Analysts and Business Analysts.
